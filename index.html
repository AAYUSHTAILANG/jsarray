Adding and Removing Elements from Array

a.from beginning i. arr.shift()=>removes first element ii. arr.unshift(10)=>adding 10 0th index *shift takes element out from the beginning, and unshift adds to the front. *others gets shifted. b.from end i. arr.pop()=>removes the last element ii. arr.push(10)=>adds at last c.from any position arr.slice(starting_index,no.of_elements_to_be_deleted,Element_to_be_inserted) takes 3 arguments out of which only 2 are mandatory let arr=[11,22,33,44,55] i.insertion only arr.slice(starting_index,0,Elements_to_be_inserted) arr.splice(2,0,10) arr=[11,22,10,33,44,55] *for entering more than 1 element arr.splice(1,0,9,8,7) arr=[11,9,8,7,22,10,33,44,55] ii.only deletion arr.splice(starting_index,no.of_elements_to_be_deleted) arr.splice(4,2) arr=[11,9,8,7,33,44,55] iii.both arr.splice(2,1,50,60,70) arr=[11,9,50,60,70,7,33,44,55]

Taking a Part of Array

arr.slice(start_index,end_index) *end index is excluded *returns new array *if end_index not mentioned then goes upto the last element. arr=[0,1,2,3,4,5,6] var slice = arr.slice(2,6) slice=[2,3,4,5] NOTE: ---- *Imagine your array is a deck of cards. splice is like rearranging the cards, either adding new ones or removing some, changing the original deck. On the other hand, slice is like taking a portion of the deck to create a new one, leaving the original deck unchanged. So, if you want to shuffle things up, use splice; if you want to make a copy, use slice. *"Sam decided to splice his collection of stickers, adding a cool one in the middle. Meanwhile, Emma chose to slice a piece of cake without changing the whole dessert, so she could enjoy it later."

Concatenation of Arrays
